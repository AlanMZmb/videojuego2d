#  Noche de Luciérnagas

##  Descripción
Este proyecto consiste en el desarrollo de un videojuego 2D utilizando la API Canvas de HTML5. El juego presenta un entorno de bosque nocturno donde el usuario debe atrapar luciérnagas en movimiento.

## Objetivo
Aplicar conceptos de animación, eventos del mouse y manipulación de objetos en Canvas para crear una experiencia interactiva.

##  Justificación
El uso de Canvas permite desarrollar aplicaciones gráficas dinámicas en el navegador, integrando lógica de movimiento, colisiones y renderizado en tiempo real.

##  Características
- 25 luciérnagas en movimiento constante
- Movimiento en múltiples direcciones
- Velocidades variables
- Colisiones con rebote entre objetos
- Eliminación de objetos mediante clic
- Reaparición aleatoria de objetos
- Contador de luciérnagas capturadas
- Fondo temático y música ambiental
- Cursor personalizado

## Instrucciones de uso
1. Observa el movimiento de las luciérnagas
2. Haz clic sobre ellas para atraparlas
3. Cada captura suma puntos
4. Las luciérnagas reaparecen en otra posición

##  Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Canvas API
- Bootstrap
